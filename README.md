# Trello API Testing Automation

## Overview
This project demonstrates comprehensive **Manual and Automated Testing** of Trello APIs, covering all CRUD operations for Boards, Lists, Cards, and Checklists.

## Features
- **Manual Testing**
  - Verified CRUD operations on Boards, Lists, Cards, and Checklists
  - Positive & Negative test scenarios
  - Screenshots from Postman

- **Automation Testing**
  - Postman scripts with JavaScript assertions
  - Console logs for real-time feedback
  - Full CRUD flow automation
  - Archive/Unarchive functionality

## How to Run
1. Clone the repository
2. Open the Postman collection
3. Set your Trello API Key, Token, and IDs
4. Run the collection
5. Check console for logs and test results

## Technologies
- Postman
- JavaScript
- Trello API
